# HTML & CSS Resume Portfolio

Professional resume-style portfolio website built using HTML5 and CSS3.

## Features
- Resume-style single page layout
- Education, Experience, Projects sections
- Hover effects and responsive design
- Hosted using GitHub Pages

## Technologies
- HTML5
- CSS3

## How to Run
1. Download or clone repository
2. Open index.html in any browser
